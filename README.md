# Анализ продаж видеоигр

## Описание проекта

Учебный проект по анализу данных о продажах видеоигр.  
В проекте рассматриваются данные об играх, платформах, жанрах, годах выпуска, оценках пользователей и критиков, а также продажах в разных регионах.

## Цель проекта

Провести предобработку данных и исследовательский анализ, чтобы выявить закономерности, влияющие на продажи видеоигр.

## Используемые инструменты

- Python
- pandas
- Jupyter Notebook
- matplotlib
- numpy

## Что было сделано

- выполнена загрузка и первичный осмотр данных;
- обработаны пропуски в данных;
- проверены и удалены дубликаты;
- преобразованы типы данных;
- проведён исследовательский анализ данных;
- проанализированы продажи по платформам, жанрам и регионам;
- построены визуализации;
- сформулированы основные выводы.

## Структура проекта

- `games_project.ipynb` — основной Jupyter Notebook с анализом;
- `README.md` — описание проекта.

## Основные выводы

В ходе анализа были изучены различия в продажах игр по платформам, жанрам и регионам.  
Также были рассмотрены оценки пользователей и критиков и их связь с продажами.

## Статус проекта

Проект завершён в рамках обучения аналитике данных.
